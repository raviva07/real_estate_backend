Real Estate Platform (Full Stack)

A full-stack Real Estate application that allows users to list, browse, and manage properties for sale or rent with role-based access control (Admin & Customer).

TECH STACK Backend Java + Spring Boot Spring Security + JWT Authentication Hibernate / JPA Swagger (OpenAPI) MySQL

Frontend React.js Axios React Router Bootstrap

Cloud Cloudinary (Image Upload)

Features

Customer Register & Login (JWT Authentication) Create Property Listings Edit / Delete Own Properties Upload Property Images (Cloudinary) View Personal Dashboard Search & Filter Properties: Location Price Range Type (Sale / Rent) Keywords View Property Details Page

Admin

View All Users Update User Roles Delete Users Approve Property Listings Delete ANY Property (Admin Privilege) View Pending Properties Manage All Properties Authentication & Authorization JWT-based authentication Role-based access: ROLE_CUSTOMER ROLE_ADMIN Secured APIs using Spring Security
